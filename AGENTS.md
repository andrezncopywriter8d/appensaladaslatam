# Project Agent Instructions

This project is **Ensaladas en Frasco**, a premium mobile-first web app for salad-in-jar recipes, dressings, weekly menus, shopping lists, conservation tips, favorites, and selling guidance for LATAM users.

## Required UI Process

1. Inspect the existing files.
2. Identify the app structure.
3. Create a short implementation plan.
4. Make scoped changes.
5. Run `npm run build`.
6. Validate the rendered mobile layout around 390px width.
7. Fix visual or functional issues before final response.

## Product Direction

- Keep the app in Spanish LATAM.
- Preserve the clean olive/cream visual identity.
- Mobile-first, centered phone-like layout on desktop.
- Use practical, simple, friendly copy.
- Avoid medical, aggressive fitness, or unrealistic weight-loss claims.
- Do not bring back Banana App, Onda Tesla, tinnitus, or generic wellness language.

## Navigation

The current navigation is:

1. Inicio
2. Buscar
3. Favoritos
4. Mi guía

## Quality Bar

- Product should feel like a real app, not an ebook.
- Home should clearly communicate salad jars, recipes, dressings, menus, tips, and favorites.
- Cards must be touch-friendly and readable around 390px width.
- Bottom nav must not cover content.
- All state should remain localStorage-based for the MVP.
